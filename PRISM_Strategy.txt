PRISM
Shared Expenses
Landing Page Strategy & Website Restructure Plan
A professional marketing direction document focused on positioning Prism as a shared 
expense tracker first, with personal finance tools as a supporting add-on.
Core Positioning Statement
Prism is a shared expense tracker built to make split costs simple, clear, and easy to settle. 
Personal finance tools are included, but they are not the main pitch.
Document Purpose
This PDF organizes the new homepage direction, page order, copy language, messaging priorities, 
and implementation prompt for restructuring the Prism website around the Shared Expenses 
feature.
Primary Goal
Reposition Prism so the first thing visitors understand is that the application solves a relatable 
everyday problem: managing shared costs, knowing who owes who, and settling balances without 
confusion.
Prism Shared Expenses Landing Page Strategy  |  Page 1
1. Marketing Pivot
New Direction
Prism should not be marketed as a personal finance app with shared expenses as a feature. 
It should be marketed as a shared expense tracker that also helps users manage personal 
finances.
Old Positioning
A personal finance dashboard with shared expenses included as one of many features.
New Positioning
A shared expense tracker that clearly shows who paid, who owes who, how much is owed, and what 
has already been settled. Personal finance tools become a valuable add-on, not the main selling point.
Why This Works




Shared costs are highly relatable across roommates, couples, friends, families, trips, and small teams.
The pain is easy to understand: people lose track of who paid, who still owes, and whether a balance 
was settled.
This gives Prism a sharper purpose than a generic budgeting or finance dashboard.
The marketing becomes more emotional and useful because it solves a real everyday money problem.
Core Website Message
Main Message
Stop guessing who owes who. Prism makes shared expenses clear.
Language Direction
The homepage should use plain language that immediately communicates the outcome. The visitor 
should feel, "This solves a problem I have."



Use: shared expenses, split costs, shared bills, who owes who, settle balances, paid back, 
reimbursements, clear records.
Reduce: budgeting software, finance management platform, spending analytics, bookkeeping system, 
advanced dashboard.
Keep the tone clear, practical, modern, and relatable.

5. Full Website Restructure Prompt
Use the prompt below to update the website direction and layout while keeping the current Prism 
visual style.
Restructure the Prism landing page so the main marketing focus is Shared Expenses, not general 
personal finance.
The new direction should position Prism as a shared expense tracker that helps users clearly see who 
paid, who owes who, how much is owed, and what has already been settled. The personal finance 
dashboard should be treated as a secondary add-on feature, not the main selling point.
Update the page order and messaging with this structure:
1. Hero Section
Lead with shared expenses. Use a strong headline like "Shared expenses, finally made clear" or "Know 
exactly who owes who." The subheadline should explain that Prism helps users track shared bills, split 
costs, view balances, and settle up clearly. Add a strong primary CTA focused on shared expenses.
2. Problem Section
Explain the common pain: shared costs get messy fast. Mention rent, groceries, utilities, trips, 
subscriptions, dinners, and group expenses. Make the user feel the problem before showing the 
solution.
3. Main Shared Expenses Section
Make this the strongest section on the page. It should clearly answer:- Who owes who?- How much?- Can I settle it?
Show a simple shared expense interface preview with clear balances, people, amounts owed, and settle 
status/buttons. Keep this section clean and easy to understand.
4. Use Cases Section
Show that shared expenses apply to roommates, couples, families, friends, trips, and small 
teams/projects. Keep the language broad and relatable.
5. How It Works Section
Use a simple 4-step flow:- Add an expense- Split the cost- See who owes who- Settle the balance
6. Personal Finance Add-On Section
Move the personal finance features lower on the page. Present them as included bonus tools: expense 
tracking, categories, spending dashboard, and financial overview. Do not make this the main focus.
7. Final CTA
End with a strong shared-expense CTA: "Start tracking shared expenses clearly."
Keep the current glassmorphism design, dark premium background, green/purple/pink gradient accents, 
and modern Prism visual style. Reduce generic personal finance language and replace it with shared
expense-focused wording such as "split costs," "shared bills," "who owes who," "settle balances," 
"paid back," and "clear records."
Make the page feel more targeted, relatable, and conversion-focused. The user should immediately 
understand that Prism solves the everyday problem of managing shared costs.
